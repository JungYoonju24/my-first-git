NUVIE (Fandom App)
View Site View Brief
신규 앱 제작 팀 프로젝트
시장 조사와 경쟁 분석을 통해 사용자 니즈를 파악하고, 이를 바탕으로 앱의 핵심 컨셉과 UX 전략을 기획했습니다.
디자인 시스템과 UI 컴포넌트는 Figma로 체계적으로 구성했으며, 주요 화면은 React 기반으로 직접 구현했습니다.
프로젝트 전반에 걸쳐 Git을 활용해 팀과 협업하며, 기획부터 인터랙션 구현까지 전 과정을 주도적으로 이끌었습니다.
UX Planning
100%
  • 경쟁사 분석 및 시장조사를 통해 차별화 요소 및 사용자 불편 사항 리서치
  • 올인원 기능 팬덤 플랫폼 방향성 수립 및 캐릭터 설정 구체화
  • 사용자의 주요 사용 흐름을 정의하여 IA 설계
UI Design
100%
  • 전체 화면 UI 설계, 컴포넌트 단위로 정리된 디자인 시스템 구축
  • 각 기능에 맞는 커스텀 아이콘과 UI 요소를 직접 디자인
  • 투표, 출석, 아티스트 카드 등 주요 섹션의 화면 흐름 정리와 인터페이스 설계
Front-End
50%
  • React를 활용해 주요 화면 구조와 컴포넌트를 구현
  • React Router를 활용한 페이지 전환 및 동적 라우팅 처리
  • Git을 통한 버전 관리 및 브랜치 전략을 기반으로 컴포넌트 단위의 UI를 실제 사용성과 맞춰 구현

Design

Process

  • WHO 포트폴리오의 유저는
    면접관과 채용 담당자
  • WHY UI/UX 디자이너로서의
    역량과 가능성 전달 목적
  • WHAT 채용자가 주목할 나의 강점과
    프로젝트     명확히 제시
  • HOW 직관적인 정보 구조와
    반응형 레이아웃
1.
BrainStorming
나를 표현할 수 있는 키워드를 도출하고,
디자인 방향의 기초를 설정했습니다.
2.
Affinity Diagram
수집한 키워드를 그룹화하여
공통된 주제와 핵심 가치를 도출했습니다.
3.
Visual Concept
포트폴리오의 메인 비주얼 스타일을 정의했습니다.
4.
Idea Sketch
콘텐츠 흐름과 인터랙션 방식을 구상하며
초기 레이아웃을 스케치했습니다.
5.
StoryBoard
사용자 관점에서 콘텐츠의 전개 흐름과
각 섹션의 역할을 시각화했습니다.
6.
Design System
일관된 사용자 경험을 위해 컴포넌트, 컬러,
텍스트 스타일 등의 시스템을 구축했습니다.
